# AishPlug Workspace

Remote fire ignition for kosher supervisors — project workspace.

## Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g. `aishplug-workspace`)
2. Upload `index.html` to the repository root
3. Go to **Settings** → **Pages**
4. Under "Source", select **Deploy from a branch**
5. Select `main` branch and `/ (root)` folder
6. Click **Save**
7. Your site will be live at `https://talpinkas.github.io/aishplug-workspace/`

## Sharing with your partner

The workspace stores data in the browser's localStorage. To sync between two people:

1. Make your changes on the site
2. Click **ייצוא JSON** (Export JSON) at the bottom — downloads a JSON file
3. Send the JSON file to your partner (WhatsApp, email, etc.)
4. Your partner opens the site and clicks **ייבוא JSON** (Import JSON)
5. Done — both of you are synced

## Features

- **סקירה (Dashboard)** — Overview metrics and latest items
- **עדכונים (Updates)** — Project timeline with tagged entries
- **שאילתות (Questions)** — Open questions board with categories, statuses, and comments
- **משימות (Tasks)** — Task board grouped by status (blocked/in-progress/to-do/done)
- **מוקאפים (Mockups)** — Interactive product UI mockups (Cook side, Mashgiach side, Alerts) in English
- **מסמכים (Documents)** — Project document registry
